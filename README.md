# HealthIsWealth

## About
---
This repository is for the repository project of a Informatics Capstone Project. The goal of this prototype is to allow patients find low-cost or free health events and cheap insurance/alternative healthcare plans. Moreover, on the side of clinicians and those who offer cheap alternatives to healthcare, they can upload their own events after going through a verification process within our system that garuntees that the events are real.

## Inclusions
---
Our project utilizes a few tools, including:
- hosting on Render
- React.JS, HTML and CSS for building

## How to Build, Deploy, and Contribute to the Codebase
---
If you would like to build and deploy with the code:
1. In any IDE, create a `git pull <this repository>`
2. Afterwards, create a branch and switch to the branch utilzing the line `git checkout -b <name of branch>`
3. Start editing!
4. In your IDE, do a `git add .`
5. Afterwards, do a `git commit -m<Message of what you changed>`
6. Lastly, `git push` to see changes on your new branch
7. Create a merge request for approval!
